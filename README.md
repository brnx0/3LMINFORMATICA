1º Salvar a aplicação na pasta "C:\xampp\htdocs"
2º Fazer o import do arquivo "3lm" no banco de dados
2ª Executar o servidor apache do xamp + MySQL 
4º Acessar pelo link http://localhost/3LMINFORMATICA/
